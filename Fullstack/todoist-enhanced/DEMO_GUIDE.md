# 🎬 Demo Guide - Todoist Enhanced

## Quick Demo Script (5 minutes)

Follow this script to showcase ALL the amazing features!

---

### 1️⃣ **First Impressions** (30 seconds)

1. Open the app → Clean, modern UI with sidebar navigation
2. **Toggle dark mode**: Click 🌙 button (bottom-right) or press `Ctrl+D`
3. Notice the smooth color transitions!

---

### 2️⃣ **Smart Task Creation** (1 minute)

1. **Click "Quick Add"** or press `Ctrl+K`
2. Type: `Finish project report @tomorrow p4`
   - Watch it auto-detect the due date (tomorrow)
   - Auto-set priority to 4 (urgent)
3. Type another: `Team meeting @next monday`
   - See it set next Monday as due date
4. Try: `Review code p2 #urgent`
   - Priority 2, with label detection

---

### 3️⃣ **Task Templates** (30 seconds)

1. **Click 📋 icon** in header (next to search)
2. Select **"📝 Daily Planning"**
3. Watch 4 tasks appear instantly!
4. Close template modal

---

### 4️⃣ **Subtasks & Progress** (45 seconds)

1. **Hover over any task** → Click ➕ button
2. Add a subtask: "Step 1 - Research"
3. Add another: "Step 2 - Implementation"
4. Notice the **progress bar** appear under the parent task
5. **Complete one subtask** → Watch progress bar update!

---

### 5️⃣ **Task Completion Celebration** (30 seconds)

1. **Complete any task** (click the checkbox)
2. Watch the **confetti animation** 🎊
3. See the **success toast** notification
4. Check your **Karma points** increase (bottom-left sidebar)

---

### 6️⃣ **Bulk Actions** (45 seconds)

1. **Hold Ctrl** and click 3-4 task checkboxes
2. Notice tasks highlight and **bulk toolbar appears** at bottom
3. Click **"✓ Complete"** → All selected tasks complete with celebration!
4. Or try **"🗑️ Delete"** to remove multiple tasks at once

---

### 7️⃣ **Drag & Drop** (20 seconds)

1. **Drag any task** up or down
2. See visual feedback (semi-transparent task, drop indicator)
3. Drop it in new position → Instant reorder!

---

### 8️⃣ **Context Menu** (30 seconds)

1. **Right-click on any task**
2. See context menu with all actions:
   - Edit, Duplicate, Add Subtask, Complete, Delete
3. Try **"Duplicate"** → Instant copy!

---

### 9️⃣ **Keyboard Shortcuts** (30 seconds)

1. Press **`?`** key → Shortcuts panel appears
2. Try **`Ctrl+K`** → Quick add task
3. Try **`Ctrl+F`** → Focus search
4. Try **`Alt+1`** → Jump to Inbox
5. Press `Escape` to close

---

### 🔟 **Kanban Board View** (45 seconds)

1. **Click 📦 icon** in header (Board View)
2. See tasks organized in **4 columns**:
   - To Do, Today, Overdue, Done
3. **Drag tasks between columns**
4. Click **☰ icon** to return to List View

---

### 1️⃣1️⃣ **Pomodoro Timer** (30 seconds)

1. Look at **top-right corner** → Pomodoro widget
2. Click **▶️** to start 25-minute timer
3. Watch countdown in real-time
4. Click **⏹️** to reset

---

### 1️⃣2️⃣ **Statistics & Insights** (45 seconds)

1. **Click "Statistics"** in sidebar
2. See productivity metrics:
   - Total tasks completed
   - Current streak 🔥
   - Karma points ⭐
3. Scroll down to **Productivity Insights**:
   - Tasks completed today
   - Overdue count
   - Average completion time
   - Most productive hour

---

### 1️⃣3️⃣ **Natural Date Suggestions** (30 seconds)

1. Click **Quick Add** (Ctrl+K)
2. **Click on the Due Date field**
3. See dropdown with quick options:
   - 📅 Today
   - ⏰ Tomorrow
   - 📆 This Weekend
   - 📍 Next Week
   - 📌 Next Month
4. Click any option → Auto-fills!

---

### 1️⃣4️⃣ **Task Details Modal** (30 seconds)

1. **Click on any task title**
2. Modal opens with:
   - Editable title
   - Due date picker
   - Priority selector
   - Description textarea
   - Subtasks list with progress
3. Make changes → Click **"Save Changes"**

---

### 1️⃣5️⃣ **Export & Import** (30 seconds)

1. **Click 📤 icon** in header
2. Choose **"Export as JSON"**
3. File downloads automatically
4. Click **📥 icon** to import
5. Select the file → Data imports!

---

### 1️⃣6️⃣ **Activity Log** (20 seconds)

1. **Click "Activity Log"** in sidebar
2. See chronological list of all actions:
   - Tasks created ✅
   - Tasks completed ✔️
   - Projects created 📁
3. Each with timestamp and details

---

### 1️⃣7️⃣ **Projects** (30 seconds)

1. **Click + button** next to "My Projects" in sidebar
2. Enter project name: "Work Projects"
3. **Pick a color** (color picker)
4. Click **Save**
5. Project appears in sidebar
6. Click project → See filtered tasks

---

## 🎯 **Power User Demo** (Advanced)

### Rapid Task Creation
```
Type these in quick succession (press Enter after each):

1. Deploy new feature @today p4
2. Code review @tomorrow p2
3. Team standup @monday p1
4. Write documentation @next week p3
```

### Template Workflow
```
1. Click 📋 → "🚀 Weekly Review"
2. All 4 tasks appear
3. Add subtasks to "Review completed tasks"
4. Complete first subtask → Watch progress bar
5. Complete parent task → Confetti! 🎊
```

### Bulk Operations
```
1. Press Ctrl+A (select all visible tasks)
2. Bulk toolbar appears
3. Complete all → Mass celebration! 🎉
4. Check Karma points skyrocket
```

## 💡 **Pro Tips for Demo**

1. **Start in dark mode** → Looks more impressive
2. **Pre-load some tasks** → More visual
3. **Show confetti first** → Hook the audience
4. **Use keyboard shortcuts** → Show power user vibes
5. **End with stats** → Show productivity gains
